The `GuildQueue` class represents a music queue for a specific guild. It manages tracks, playback, voice connections, and audio processing.

## Getting a Queue

```typescript
import { useQueue } from 'discord-player';

const queue = useQueue(guild.id);
// or
const queue = player.nodes.get(guild.id);
```

## Playback Control Methods

### play()

Start playing a track:

```typescript
await queue.play(track);
```

<ParamField path="track" type="Track | null" required>
  The track to play, or null to play next in queue
</ParamField>

<ParamField path="options" type="PlayOptions">
  Play options (seek, immediate, filtersUpdate, encoderArgs)
</ParamField>

### pause()

Pause playback:

```typescript
queue.node.pause();
```

<ResponseField name="returns" type="boolean">
  True if paused successfully
</ResponseField>

### resume()

Resume playback:

```typescript
queue.node.resume();
```

<ResponseField name="returns" type="boolean">
  True if resumed successfully
</ResponseField>

### skip()

Skip the current track:

```typescript
const skipped = await queue.node.skip();
```

<ResponseField name="returns" type="Promise<boolean>">
  True if skipped successfully
</ResponseField>

### stop()

Stop playback and clear queue:

```typescript
queue.node.stop();
```

### seek()

Seek to a position:

```typescript
await queue.node.seek(60000); // Seek to 1 minute
```

<ParamField path="position" type="number" required>
  Position in milliseconds
</ParamField>

<ResponseField name="returns" type="Promise<boolean>">
  True if seek was successful
</ResponseField>

## Track Management Methods

### addTrack()

Add a track to the queue:

```typescript
queue.addTrack(track);
```

<ParamField path="track" type="Track | Track[]" required>
  Track or array of tracks to add
</ParamField>

### insertTrack()

Insert a track at a specific position:

```typescript
queue.insertTrack(track, 0); // Insert at front
```

<ParamField path="track" type="Track | Track[]" required>
  Track or array of tracks to insert
</ParamField>

<ParamField path="index" type="number" default="0">
  Position to insert at
</ParamField>

### removeTrack()

Remove a track:

```typescript
const removed = queue.removeTrack(2); // Remove track at index 2
// or
const removed = queue.removeTrack(track); // Remove specific track
```

<ParamField path="track" type="TrackResolvable" required>
  Track, track ID, or index
</ParamField>

<ResponseField name="returns" type="Track | null">
  The removed track
</ResponseField>

### clear()

Clear all tracks from queue:

```typescript
queue.tracks.clear();
```

### shuffle()

Shuffle the queue:

```typescript
queue.tracks.shuffle();
```

<ResponseField name="returns" type="Track[]">
  The shuffled tracks
</ResponseField>

## Example

Complete queue management example:

```typescript
import { useQueue } from 'discord-player';

// Get the queue
const queue = useQueue(interaction.guildId);

if (!queue) {
  return interaction.reply('No music is being played!');
}

// Current track info
const track = queue.currentTrack;
console.log(`Now playing: ${track.cleanTitle}`);
console.log(`Duration: ${track.duration}`);

// Playback control
queue.node.pause();
queue.node.resume();
await queue.node.skip();
await queue.node.seek(30000);
queue.node.setVolume(75);

// Queue management
queue.addTrack(newTrack);
queue.insertTrack(track, 0);
queue.removeTrack(2);
queue.tracks.shuffle();
queue.tracks.clear();

// Repeat modes
queue.setRepeatMode(QueueRepeatMode.QUEUE);

// History
const prev = queue.history.previousTrack;
await queue.history.back();

// Filters
await queue.filters.ffmpeg.toggle('bassboost');
queue.filters.equalizer.setEQ([
  { band: 0, gain: 0.25 },
  { band: 1, gain: 0.25 }
]);

// Connection
const timestamp = queue.node.getTimestamp();
const bar = queue.node.createProgressBar();

console.log(`${timestamp.current.label} ${bar} ${timestamp.total.label}`);
```
